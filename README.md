# Spring-Forum
Java Spring 프레임워크로 제작한 간단한 게시판

## 개발 환경
---
- Spring Framework
- JDK 1.8
- Tomcat v8.5
- MyBatis + Oracle Database
- Bootstrap

## 프로젝트 기능
---

- 인증 기능 (회원가입, 로그인, 로그아웃)
- 정보 수정 기능
- 게시물 등록, 수정 및 삭제 기능
- 댓글 등록, 수정 및 삭제 기능
- 파일 첨부, 수정 및 삭제 기능
